# VORTEX LIMITED — React port

This repository is a React (Vite) port of the original static site.

Quick start:

```bash
npm install
npm run dev
```

Then open the URL shown by Vite (usually http://localhost:5173).

Notes:
- Images remain in the `images/` folder and are referenced from `/images/...`.
- External icon fonts and ScrollReveal are loaded from CDNs in `index.html`.
